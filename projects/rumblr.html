Rumblr

clone of the popular micro-blogging site Tumblr

navigate_next

This is an ongoing project to build a Tumblr clone in Rails simply as an exercise. While not 100% completed, many of the core features are implemented. Users can create an account, create posts (images, texts, and animated GIFs), reblog posts, and follow other users.

Rails, Paperclip, Devise, HTML, CSS, BootStrap
