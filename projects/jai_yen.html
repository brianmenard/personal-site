DuJour

recipe management tool

navigate_next

Dujour is a recipe management tool in the vein of AllRecipes. Dujour uses Devise and/or OmniAuth for authentication and Paperclip for photo upload, allowing users to create and save recipes, which will then be shared with the community.

Rails, ActiveRecord, Devise & OmniAuth, Paperclip, BootStrap
